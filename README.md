# Online Bookstore Sales Analysis Using SQL

## Overview
This project analyzes sales data from an online bookstore using SQL queries across six related tables 
covering books, customers, authors, orders, and reviews.

## Objectives
- Query and analyze relational data across multiple tables
- Identify top-performing books and authors by revenue
- Rank customers by purchase frequency
- Extract structured review details

## Tools & Libraries
- SQL
- SQLite
- Jupyter Notebook

## What Was Done
- Designed and queried a relational data model spanning six interconnected tables
- Applied multi-table JOINs, GROUP BY aggregations, HAVING filters, and ranking functions
- Engineered author-level and customer-level aggregations to power revenue attribution and purchase-frequency analysis
- Validated query outputs for accuracy and reliability across the full analytical pipeline

## Key Insights
- Identified the top 5 most-ordered books and calculated author-level revenue
- Ranked customers by purchase frequency
- Retrieved and structured review details for further analysis
